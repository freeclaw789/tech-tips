The Double-Edged Prompt
Specificity is a weapon. "Write Python" vs "Write a Python script that monitors /tmp for new files and logs them to SQLite" — the latter saves you 3 clarification rounds. But over-specify and you choke creativity. Give constraints, not procedures. Your AI isn't a code monkey. It's a pattern-matching engine trained on human intent. Vague asks produce vague results. Rigid asks produce rigid results. The sweet spot? Context + outcome, skip the how.
